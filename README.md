# Universal Convertor

A simple web-based tool for converting between various units, currencies, and data formats.
> [https://nayanpanchal2007.github.io/Universal-convertor/]

## Features
- Unit conversions (length, weight, temperature, etc.)
- Currency exchange rates
- Data format conversions (JSON/XML, CSV/Excel, etc.)
- Responsive design for mobile and desktop

## Usage
1. Open `index.html` in any modern web browser
2. Select the conversion type from the menu
3. Enter the input value and choose target format
4. View the converted result instantly

## Installation
For local development:
```powershell
# No build tools required - just open in browser
Start-Process -FilePath "chrome.exe" -ArgumentList "file://$(Get-Location)/index.html"
```
